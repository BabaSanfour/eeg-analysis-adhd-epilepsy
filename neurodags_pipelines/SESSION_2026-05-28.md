### 3. Fixed critical channel-positions bug

**Problem:** All 19 EEG channels had NaN positions in source `.vhdr` files. The old pipeline loaded positions from the BIDS `*_electrodes.tsv` sidecar; the new pipeline did not.

**Consequences:**
- RANSAC (`ransac_bad_channels`) silently failed — exception caught by bare `except (ValueError, OSError): pass`, so no bad channels ever detected
- AutoReject in `residual_denoise` crashed with `RuntimeError: Valid channel positions are needed`
- ICA in `ica_artifact_correction` crashed with `ValueError: ica.n_components (20) cannot be greater than len(picks) (19)`

**Fixes applied:**

`nodes_annotations.py` — `inject_block_annotations` now reads `*_electrodes.tsv` after injecting block annotations:
```python
elec_candidates = sorted(raw_path.parent.glob("*_electrodes.tsv"))
if elec_candidates:
    elec_df = pd.read_csv(elec_candidates[0], sep="\t")
    if {"name", "x", "y", "z"}.issubset(elec_df.columns):
        ch_pos = {row["name"]: [float(row["x"]), float(row["y"]), float(row["z"])]
                  for _, row in elec_df.iterrows()
                  if pd.notna(row["x"]) and pd.notna(row["y"]) and pd.notna(row["z"])}
        if ch_pos:
            montage = _mne.channels.make_dig_montage(ch_pos=ch_pos)
            raw.set_montage(montage, on_missing="ignore")
```

`nodes_ica.py` — clamped n_components:
```python
picks = _mne.pick_types(raw.info, eeg=True, exclude="bads")
n_components = min(n_components, len(picks))
```

`nodes_qc.py` — `_strip_bad_annotations` used setter on read-only property:
```python
# was:  raw_clean.annotations = raw_clean.annotations[keep]
raw_clean.set_annotations(raw_clean.annotations[keep])
```

### 4. Implemented descriptor QC node

New file: `neurodags_pipelines/nodes_descriptor_qc.py`

Node `generate_descriptor_qc_record`:
- Takes path to any one NC file in the condition features directory
- Scans all sensor-level NC files (AbsBandPower, RelBandPower, LogBandPower, Corrected variants, BandRatios, FooofScalarsDs, FooofPeaksDs, HjorthParams, all complexity descriptors)
- Scans all pooled NC files (band power + band ratios + FooofExponent/Offset/RSquaredPooled)
- Builds flat epoch × feature DataFrames with column names matching old convention:
  - `band_{subtype}_{band}_ch-{space}` / `_chgrp-{region}`
  - `param_{metric}_ch-{space}`
  - `complexity_{name}_ch-{space}`
- Skips `fooof_aperiodic_knee` (always NaN when `aperiodic_mode='fixed'`)
- Calls `run_descriptor_subject_qc` from `eeg_adhd_epilepsy/qc/descriptor_qc.py`
- Generates HTML report + CSV summaries in `qc/` subdirectory
- Returns JSON NodeResult with QC summary row

Wired into `step-1_pipeline@extraction.yml`:
```yaml
DescriptorQCRecord:
  overwrite: False
  for_dataframe: False
  nodes:
    - id: 0
      derivative: AbsBandPower.nc
    - id: 1
      node: generate_descriptor_qc_record
      args:
        nc_file: id.0
        condition_name: $condition_name
```

HTML reports land at:
`{deriv}/descriptor_qc_reports/sub-{id}/ses-01/descriptor_qc/sub-{id}_ses-{sess}_{condition}_descriptor_qc_report.html`
